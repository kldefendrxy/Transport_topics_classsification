# TG Transport Classifier

Классификация сообщений Telegram-каналов (автотранспорт vs метро) с использованием MLOps-пайплайна.

## Цель
Создать модель, которая по тексту сообщения определяет, относится ли оно к автомобильной тематике или к метрополитену.

## Основные этапы
1. Сбор данных из Telegram (Telethon)
2. Очистка и лемматизация текста
3. Векторизация и обучение моделей (TF-IDF, LightGBM, DistilBERT)
4. Версионирование данных через DVC
5. Логирование экспериментов в MLflow
6. Деплой через FastAPI в Docker
